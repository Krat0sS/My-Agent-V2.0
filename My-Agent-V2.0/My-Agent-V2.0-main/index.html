My-Agent V2.0

My-Agent V2.0 — 单脑决策智能体

route() 唯一决策 · 四路径分发 · 四层记忆闭环

💡 核心理念
单脑决策：route() 是唯一决策入口，不再有左右脑打架的问题。

四路径分发：技能命中 → 模板匹配 → 任务分解 → 纯对话，优先级明确。

四层记忆：工具效果（越用越准）→ 技能沉淀（自动学习）→ 用户偏好 → 系统优化。

安全第一：命令白名单、路径检查、频率熔断、审计日志，默认开启。
🧠 单脑决策
route() 唯一决策入口，四路径分发
🔧 工具注册表
58 个工具，自动发现，动态 Schema，降级链
🧠 意图路由
BM25 + LLM 精排，分类→匹配→执行→沉淀
🎯 技能系统
首次走分解流程，二次极速执行，TTL 暂存
🌐 浏览器自动化
Playwright 驱动，操作后自动返回页面上下文
📊 执行日志
全链路打点，工具效果评估，为进化提供数据
🔒 安全模块
文件系统守卫 + 命令拦截 + 敏感文件保护
🔄 Git 工具链
改代码→测试→提交→推送，测试门禁防误推
🛠️ 技术栈
Python 3.10+ DeepSeek API Flask Playwright GitPython SQLite Streamlit jieba
🚀 启动方式
双击运行 启动.py，自动完成：
  1️⃣ 检测 Python 环境
  2️⃣ 创建虚拟环境
  3️⃣ 安装所有依赖（含 pytest、Playwright、GitPython）
  4️⃣ 环境检查 → 进入菜单选择模式
🔮 未来方向
让 Agent 自我升级：浏览网页 → 下载代码 → 读写修改 → 跑测试 → 推 GitHub。

三个瓶颈按优先级：
  1. 浏览器反馈可靠性 ← 已基本解决
  2. LLM 判断力 → 用测试结果代替 LLM 自评
  3. 安全性 → 测试通过门禁，不需要全开安全层
